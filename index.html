Master Class

Master Class is a JavaScript library that allows you to target mobile browser functionality in your stylesheet. It's designed to be used on websites that support many types of mobile browsers.

For example, you can use Master Class to show a high resolution images to devices with a high pixel density like the iPhone4:

/* Show the high resolution image if the device has a high pixel density, like the iPhone4: */
.hd #image {
  background: url(high-resolution.png); 
  background-size: 100%; 
}

/* Otherwise show a normal resolution image: */
.no-hd #image { 
  background: url(normal-resolution.png); 
}

To do this, Master Classes adds classes to the <html> that allows you to target browsers by:

  • JavaScript Support
  • Touch Support
  • Mobile OS
  • Orientation
  • Pixel Density

The current Master Class is: <html class="">

Using Master Class

Master Classes works by adding classes to the that allows you to browsers with specific properties.

  1. Include the Master Class JavaScript file in page.
  2. Add the default master classes to the html element of your page:
    <html class="no-js no-os no-touch no-hd">

The following master classes are availible:

  • JavaScript Support: no-js, js
  • Mobile OS: no-os, ios, android, nokia, blackberry
  • Orientation: portrait, landscape
    Orientation classes are only added to devices that support the window.orientation property.
  • Pixel Density: no-hd, hd, hd15, h20
    The hd class is added to all high pixel density devices, while hd15, hd20 are added to devices with specific pixel densities.
